## ğŸ” Web Login Integration (Key Feature)

Qutopia includes a lightweight web-based login system designed to connect the game with an external database.

**How it works:**

- Players log in using their **NIS (Student ID)** before starting the game.
- If the NIS is registered in the web system, the game automatically retrieves and displays the player's name.
- If the NIS is not registered, the game still allows access and displays the entered ID.
- Player data is stored in the web system, enabling persistent tracking.

> This feature highlights my ability to integrate gameplay with external web services and handle real-time data communication between a game client and a backend system.
